# Agent Startup Skills 🚀

A collection of open-source "Skills" that give your AI coding assistants (like Antigravity, Claude Code, and Codex) the superpower to act as a complete Startup Accelerator.

## Included Skills

- **`saas-accelerator`**: Transforms your AI into a 3-part multi-agent team (Researcher -> Coder -> Reviewer) that pauses for your executive approval before writing code.

## How to Install

### For Antigravity / Claude Code
1. Copy the `skills/saas-accelerator` folder.
2. Paste it into your local AI configuration directory (e.g., `~/.gemini/antigravity/skills/` or `~/.claude/skills/`).
3. The AI will now automatically trigger the SaaS Accelerator workflow whenever you ask it to build a startup feature!
